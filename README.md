**Prediction with Regression**
This project is about predicting values using Simple Linear Regression.
I have worked on two different cases: Delivery Time and Salary Hike.
________________________________________________________________________________
**🛠 Tools Used**
**Python**: For coding and building models.
**Libraries**: Pandas (Data), Seaborn/Matplotlib (Graphs), Scikit-learn (ML).
**Excel/Google** Sheets: For quick calculations and charts.
**Word & PPT**: For reports and presentations.
__________________________________________________________________________________
**📦 Project 1: Delivery Time Prediction**
**Goal**: Predict "Delivery Time" based on "Sorting Time".
What I found (Insights):When Sorting Time increases, Delivery Time also increases.
The model accuracy ($R^2$) is 0.68, which is a medium-level success.
**Benefit**: It helps delivery companies tell customers when their package will arrive.
__________________________________________________________________________________
**💰 Project 2: Salary Hike Prediction**
**Goal**: Predict "Salary" based on "Years of Experience".
What I found (Insights):There is a very strong connection between experience and salary.
The model accuracy ($R^2$) is very high (above 0.95), which means it is very reliable.
**Benefit**: It helps HR teams decide the right salary for a new employee based on their experience.
